# Installation

## npm

Use NPM for installation.

```shell
npm i slack-ui -S
```

## CDN

How to use CDN.

```html
<!-- Import style -->
<link rel="stylesheet" href="https://unpkg.com/slack-ui/lib/theme-chalk/index.css">
<!-- Import component -->
<script src="https://unpkg.com/slack/lib/index.umd.js"></script>
```
