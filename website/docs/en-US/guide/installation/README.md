# Installation

## npm

Use NPM to install.

```shell
npm i slack-ui -S
```

## CDN

Use whit CDN

```html
<!-- import style -->
<link rel="stylesheet" href="https://unpkg.com/slack-ui/lib/theme-chalk/index.css">
<!-- import js -->
<script src="https://unpkg.com/slack/lib/index.full.js"></script>
```
